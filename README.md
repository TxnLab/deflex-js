# @txnlab/deflex (Archived)

> ⚠️ **This repository has been archived and is read-only.**
>
> The Deflex Order Router has been rebranded to **Haystack**.

## New Repository
**[TxnLab/haystack-js](https://github.com/TxnLab/haystack-js)**

## New Package
**[@txnlab/haystack-router](https://www.npmjs.com/package/@txnlab/haystack-router)**

## Quick Migration

```bash
npm uninstall @txnlab/deflex
npm install @txnlab/haystack-router
```

## Breaking Changes

The package and class names have changed:
- Package: `@txnlab/deflex` → `@txnlab/haystack-router`
- `DeflexClient` → `RouterClient`
- `DeflexConfig` → `Config`
- `DeflexQuote` → `SwapQuote`
- `DeflexTransaction` → `SwapTransaction`

All functionality remains identical.
